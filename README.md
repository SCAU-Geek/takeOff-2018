# weather.pyw

> 告诉我明天天气可以吗？

可以。

## 这是什么？
这是用Python3写的一个获取广州市天气的脚本，每隔一个小时，便会通过macOS或者Windows 10内建的「通知中心」汇报明天的天气状况。

## 怎么使用它？

 1. 确保系统有Python 3。
 2. 直接双击weather.pyw，就可以让它在后台静默运行。如果是Windows用户，可能需要安装额外的`win10toast`依赖项，在命令行用pip命令安装即可：
	`pip install win10toast`

## 它是如何工作的？
有很多种方式可以获取天气信息，如：

 - 通过开放平台API
 - 通过爬取气象站数据

因为必须注册账号并获取App ID才能通过限制颇多的API来获取数据，于是便决定爬[中央气象台](http://www.nmc.cn/publish/forecast/AGD/guangzhou.html)的网页，用Python3自带的`HTMLParser`解析数据，然后调用系统API，把处理好的数据显示出来。

## 为什么要做成这样？
我觉得没必要看个天气还得打开浏览器。让系统自动提醒你可能比较优雅。何况你可以随时可以打开通知中心，查看明天的天气简报。

## 计划添加的内容
 - 根据明天的天气状况，选择对应的通知图标。
 - 如果当前时间是凌晨，则显示当天的天气信息。
 - GNU/Linux支持。
